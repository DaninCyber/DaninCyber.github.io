# CyberSec Insights Blog

A Jekyll-powered cybersecurity blog hosted on GitHub Pages.

## Quick Start

1. Clone this repository
2. Update `_config.yml` with your information
3. Replace "yourusername" in links with your actual username
4. Add your posts to `_posts/` folder
5. Push to GitHub

## Writing New Posts

Create a new file in `_posts/` with the format: `YYYY-MM-DD-title.md`
```markdown
---
layout: post
title: "Your Post Title"
date: 2025-11-20
category: Category Name
tags: [tag1, tag2, tag3]
---

Your content here...
```

## Local Development
```bash
bundle install
bundle exec jekyll serve
```

Visit http://localhost:4000
